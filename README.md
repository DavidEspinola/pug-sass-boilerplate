# Pug Sass boilerplate

## Usage
Install dependencies:
`npm install`

Build project:
`npm run build`

Serve in development mode (recompilation and live-reload):

```npm run dev:build```

```npm run dev:serve```

(Need 2 separate terminal tabs)
